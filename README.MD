# 🚀 Affine Encrypter/Decrypter

Affine şifreleme algoritmasının Python dilinde hem şifrelenmesi hem de çözülmesini sağlayan bir program.

## Affine Algoritması
* **Çarpımsal ($a$)** ve **Toplamsal ($b$)** anahtarları kullanarak metin şifreleme.
* Şifreli metni orijinal haline döndürmek için çözme (decryption) işlevi.
* Anahtar $a$'nın mod $26$'daki tersini (**modüler ters**) hesaplama yeteneği.
